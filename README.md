# Laboration 2-Frontend

## https://lab2frontend.netlify.app/

Denna webbapp arbetar i samband med en webservice från Render och en MySQL databas från Aiven.
Den kör asynkrona fetch-anrop för att invänta svar från APIn skapad, och läser sedan automatiskt ut innehållet (om det finns) på förstasidan.
Webbapplikationen kan även användas för att radera poster, lägga till nya och uppdatera befintliga.

En about-sida beskriver webbplatsen, likt denna README.
